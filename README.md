# AgenticAISurveyDetection
This repository includes a guide and the code to detect the use of Comet Agentic AI browsers to take Qualtrics surveys.

Watch this video for an overview of how Comet is being used to take surveys for humans - https://www.loom.com/share/e8c169f665b14f648f2e25c5808944cf

This video demonstrates how to insert the code into Qualtrics in order to detect the use of a Comet browser - https://www.loom.com/share/b077fd09a1094f848117a43076c199e4
